GitHub + Hexo搭建个人免费博客网站

本文介绍如何使用GitHub + Hexo搭建个人免费博客网站，所有内容亲测，绝对可用。

一、准备工作

GitHub账号

需要有一个GitHub账号，没有的话到 官网 申请一个。
注册很简单，不懂的话可以参考 GitHub申请账号

安装Git

在自己电脑上安装好Git，hexo部署到GitHub时要用。
网上找篇教程或者参考 Git安装。

安装NodeJS

在自己电脑上安装好NodeJS，Hexo是基于NodeJS编写的，所以需要安装NodeJS和npm工具。
网上找篇教程或者参考 NodeJS安装及配置。

二、创建仓库

在GitHub上创建一个新的代码仓库用于保存我们的网页。

点击Your repositories，进入仓库页面。

点击New按钮，进入仓库创建页面。

填写仓库名，格式必须为<用户名>.github.io，然后点击Create repository。

点击creating a new file创建一个新文件，作为我们网站的主页。

新文件的名字必须为index.html，内容先随便写一个简单的，内容示例如下，填写之后点击Commit new file提交。

GitHub 中找到我们主页的地址为 https://<用户名>.github.io/

浏览器中访问，展示成功。

这里创建的网页是非常简单的，只是为了演示下GitHub 的使用方式。

三、发布到GitHub

使用 SSH 密钥认证
生成 SSH 密钥（如果之前没生成过）

把公钥添加到 GitHub 账户
复制~/.ssh/id_rsa.pub文件的内容。
登录 GitHub，依次点击 Settings→SSH and GPG keys→New SSH key，粘贴公钥。
更新 Hexo 配置文件
把 deploy 部分的 repo 修改为 SSH 格式：

编辑 nano _config.yml

验证 SSH 连接

bash
ssh -T git@github.com

要是看到Hi <用户名>! You’ve successfully authenticated…，就说明连接成功了。

五线谱

设置

hexo-tag-abcjs

Helps you to render standard music notation with abcjs on your blog posts

Install
Configuration

You can configure this plugin in _config.yml.

  • js - The abcjs library url. default: /js/abcjs_midi_5.9.1-min.js
  • css - The abcjs style url. default: /css/abcjs-midi.css
  • midi - Whether enable midi audio. default: true
  • animation - Whether enable absjs animation. When true, highlights the current notation. default: true
  • animationColors - The animated notation color. default: ['#000000', '#3d9afc']
  • options - The abcjs Configuration. default as above
  • scriptId - The id of inserted script tag. default: hexo-tag-abcjs
Usage
  • autoplay - Enable autoplay
  • no-midi - Disabled midi audio
  • no-animation - Disabled audio animation
