Gitalk

Gitalk 是一个基于 Github Issue 和 Preact 的现代评论组件。

  1. 单击此处注册新的 OAuth 应用程序。其他内容可以随意填写，但一定要填写正确的回调 URL（通常是评论页对应的域名）。然后，您将获得一个 Client ID 和一个 Client secret。

  2. 创建一个仓库，将 Gitalk 评论存储在 GitHub 中。

  3. 将值 enable 设置为 true ，在步骤 1 中添加 Client ID（客户端 ID client_id ）和客户端密钥（ client_secret ），在步骤 2 中添加您的 Github 用户名（ github_id 和 admin_user ）和创建的仓库名称（ repo ），并在 NexT 配置文件的 section 中 gitalk 编辑其他配置，如下所示：
